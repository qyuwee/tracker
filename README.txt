Код основной программы содержится в файле main.py.
Базовые функции описаны в файле function_guide.py.

Руководство по использованию:



Настройка FPS

Для корректной работы требуется указать значение FPS вашей камеры:

Раскомментируйте закомментированные строки в main.py (отмечены в коде).

Запустите программу — в левом верхнем углу отобразится текущий FPS.

Впишите это значение в параметр fps и закомментируйте строки снова.



Параметр alpha

Влияет на скорость обработки: чем больше значение (до 1.0), тем быстрее обновляется скорость, но снижается плавность.

Рекомендуемое значение: 0.325 (оптимальный баланс).



Выбор трекера

По умолчанию используется CSRT (точный, но ресурсоёмкий). Альтернативы:

GOTURN — быстрее, но требует предобученной модели.

Другие (например, KCF, MOSSE) — для экспериментов.



Определение выхода за границы

Переменные left, right, top, bottom:

Если True — объект вышел в соответствующем направлении.

dist_x и dist_y — расстояние (в пикселях) от объекта до границы прямоугольника.



Важно

При быстром движении объекта данные могут обновляться с задержкой (особенно на низком FPS).

Для точности увеличьте FPS или используйте трекер KCF/MOSSE.
