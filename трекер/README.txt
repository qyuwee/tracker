скачать все зависимости сразу:
pip install -r requirements.txt

Есть две версии модели
одна для слабого по, другая для нормального и сильного.
Для слабого нужно импортить всю папку best_openvivo_model
Для другого просто best.pt
При импорте использовать raw строки (просто добавить символ r перед ковычками)
MODEL_PATH = r'путь/к/модели'

Управление:
1 - показать интерфейс
2 - скрыть интерфейс
c - сбросить отлсеживаемые объекты
q - закрыть программу
+/= - увеличить порог уверенности на 0.05
- - уменьшить порог уверенности на 0.05
